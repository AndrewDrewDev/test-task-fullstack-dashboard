# Features/Stack: 

- Figma
- Typescript only;
- Responsive layout design;
- SCSS;
- React + Redux/Thunk;
- NodeJS + Express;

# How Start:

```shell
# Install dependencies

npm run dep:install

# Start Client and Server
npm run dev
```
